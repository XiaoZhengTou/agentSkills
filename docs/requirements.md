# 贷后管理产品线需求文档

## 项目概述

使用 Next.js 全栈框架开发贷后管理产品线系统，覆盖个贷、车贷等不良资产管理场景。

## 产品背景

贷后管理系统用于金融机构对已发放贷款进行全生命周期管理，重点处理逾期、不良资产的催收、核销、处置等业务流程。

## 用户角色

- 管理员（Admin）：拥有全部权限，负责系统配置、用户管理、数据总览

## 核心业务流程

```
贷款导入 → 逾期识别 → 案件创建 → 催收跟进 → 还款登记 → 结案/核销
```

### 各阶段说明

1. 贷款导入：批量导入个贷/车贷合同数据
2. 逾期识别：系统自动标记逾期账户，生成不良案件
3. 案件管理：查看案件详情、借款人信息、还款历史
4. 催收跟进：记录每次催收动作（电话、上门、短信等）
5. 还款登记：录入还款金额、日期、方式
6. 结案/核销：正常结清或申请核销处理

## 核心业务模块

- 仪表盘：逾期总览、回款统计、案件趋势图
- 案件管理：个贷案件列表、车贷案件列表、案件详情
- 催收记录：跟进记录、联系历史
- 还款管理：还款登记、还款台账
- 客户管理：借款人信息、联系方式
- 报表中心：逾期率、回收率、催收效率报表
- 系统管理：用户管理、权限配置、数据字典

## 技术选型

- 框架：Next.js 14（App Router，全栈）
- 语言：TypeScript
- 数据库：PostgreSQL + Prisma ORM
- UI 组件库：shadcn/ui + Tailwind CSS
- 图表：Recharts
- 认证：NextAuth.js
