# wireframe-generate

## description
根据 PRD 中的功能需求和用户故事，生成低保真线框图。输出 ASCII 原型图 + 组件布局说明 + 响应式断点方案，供前端开发和设计评审使用。

## steps
1. 读取 `.claude/project-context.json`，若不存在则先执行 `/detect-project`
2. 读取 `.claude/handoffs/prd.md`，提取页面列表和功能模块
3. 对每个页面：
   - 确定页面类型（列表页 / 详情页 / 表单页 / 仪表盘）
   - 规划信息层级和主要区块
   - 绘制 ASCII 线框图（桌面端 + 移动端）
   - 标注交互热区和状态变化
4. 输出响应式断点方案（sm / md / lg / xl）
5. 列出每个区块对应的组件名称，与 component-design 衔接
